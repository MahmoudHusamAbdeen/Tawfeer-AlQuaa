# Tawfeer Source Code

This folder contains the Tawfeer project source code.

The source code is uploaded as:

Tawfeer_Source_Code.zip

The project includes the main Tawfeer system components:

1. User interface
2. Food donation flow
3. Food request flow
4. AI food reuse feature
5. Admin dashboard
6. Driver interface
7. Government dashboard

Reviewers can download and extract the ZIP file to inspect the project files.
